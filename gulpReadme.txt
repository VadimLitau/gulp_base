Описание пакетов

gulp -сам гулп

gulp-sass - сборка сасс,сццц

sass - сам сцсс

gulp-file-include - подключение файлов друг в друга html inlude 

gulp-clean - удаление файлов

gulp-server-livereload - сервер с автообновлением страницы

gulp-sourcemaps - исходные карты для цсс

gulp-plumber - фикс ошибок при сборке

gulp-notify - нотификация

gulp-group-css-media-queries - группирорвка цсс медиа запросов 

для работы с гулп надо установить глобально командную строку гулп
npm install --global gulp-cli
